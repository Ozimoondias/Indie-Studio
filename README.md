## Usage

Use Z,Q,S,D for move and ENTER for fire... **obviously**
<br>
And **_Arrows_** for player 2
<br>

## Dependencies
* Irrlicht
* Irrklang

## Compile and run

Compile with a C++11 compliant compiler:
```sh
cd Install
sudo sh compile.sh
./cpp_rtype
```
